Thank you for choosing Evalo - Minimal Saas Startup and Agency Html Template

Evalo is a clean, minimal and stylish template for all kind of agency, startup and companies. Any one can use Evalo to build their website for It startup, software company, mobile app business, app landing page, portfolio and more.

Evalo offers a beautiful, professional design that is clean and simple. Evalo is responsive and will look great at all screen sizes. This Responsive HTML Template has been built for the user who wants to showcase their business in an elegant fashion.

Evalo is a great html template perfect for your next website. It is fully responsive and retina ready for high quality layout with excellent performance. Evalo comes with 8 home page variations so you can build the perfect look and feel for your company. Evalo has been tested on Chrome, Safari, Firefox and Opera.

Evalo is very easy to customize and all section is well commented. You can easily manipulate the code and layer order.
